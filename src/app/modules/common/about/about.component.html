Astronomy Picture of the Day (APOD) Data is fetched from the official NASA API. The APOD can also be a video and npm module safe-pipe was required to prevent the app from crashing. Data is displayed in a Bootstrap card with a description, date and credit text (where available) below.
Price data for a range of cryptocurrencies is fetched from the CryptoCompare API and is displayed using a Bootstrap responsive table. The table also shows svg icons for each currency, retrieved from the assets folder. The cryptocurrency prices are displayed in euro but this can be changed to other currencies.
User location data is fetched from ipapi, a real-time Geolocation & Reverse IP Lookup REST API. If location data is not available then the coordinates of the Eiffel Tower are used to center the map. Map data is obtained using Leaflet, an open-source JavaScript library for mobile-friendly interactive maps that requires no API key.
News is fetched from the New York Times Top Stories HTTP REST API.
Data on books is fetched from the Google Charts APIChart data is displayed in a grid of Bootstrap cards.
Data on books is fetched from the Google Books APIBook data is displayed in a grid of Bootstrap cards. Clicking on a card will activate ruote the user to a page with more details about the book.
Data on bestsellers is fetched from the New York Times APIBestseller data is displayed in a grid of Bootstrap cards. Clicking on a card will route the user to a page with more details about the book.
User and repository data is fetched from the Github REST API v3.Github card shows Github repo details from user search. Repo button redirects to repo list page.
